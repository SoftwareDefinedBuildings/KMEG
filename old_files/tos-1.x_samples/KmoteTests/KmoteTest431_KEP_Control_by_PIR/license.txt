+ Sensing Scale

1. Sensing Data (12 bit)
   0x000 means no Electric Device is loaded
   OxFFF means 10A Electric Device is loaded
   if 3.3A current flowing, might be the value is 0x555 
   if 6.6A current flowing, might be the value is 0xAAA
